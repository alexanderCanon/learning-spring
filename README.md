# learning-spring
Just learning spring, using Spring MVC and JPA

## Technologies Used
- **Java 17+**
- **Spring Boot**
- **Spring MVC**
- **Spring Data JPA**
- **Maven**

## Getting Started
1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/learning-spring.git
   ```
2. **Build the project**:
   ```bash
   mvn clean install
   ```
3. **Run the application**:
   ```bash
   mvn spring-boot:run
   ```

## Features
- Implementation of the **Model-View-Controller** (MVC) pattern.
- Data persistence and ORM management with **JPA/Hibernate**.
- Configuration of REST controllers and service layers.
- Integration with H2/MySQL for data storage.
